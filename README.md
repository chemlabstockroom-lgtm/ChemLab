# ChemLab
A Web-Based Chemistry Laboratory Inventory Management System for DLSU-D. Features barcode-integrated borrowing, real-time tracking, and automated experiment-based material lists to replace manual Excel processes and reduce administrative errors.

ChemLab Inventory Management System (CIMS)
Automating Chemistry Laboratory Operations at De La Salle University - Dasmariñas

📌 Background
The current chemistry stockroom at DLSU-D relies on manual Excel files and physical folders, leading to data discrepancies, slow borrowing procedures, and inefficient tracking.

ChemLab IMS is a web-based solution designed to digitize these workflows. By integrating barcode scanning and experiment-based borrowing, the system eliminates manual errors and provides real-time oversight of laboratory resources.

✨ Key Features
Real-Time Inventory Tracking: Monitor stock levels of chemicals and equipment instantly.

Experiment-Based Borrowing: Students can request materials based on specific laboratory experiment lists via QR/Barcodes.

Barcode Integration: Streamlined check-out and return process to minimize human error.

Automated Reporting: Generate usage and inventory reports for better purchasing decisions.

User-Friendly Interface: Dedicated portals for students, faculty, and stockroom staff.

🛠 Tech Stack
Frontend: React.js / Vite

Backend: Node.js & Express

Database: MongoDB (MERN Stack)

Authentication: JWT (JSON Web Tokens)

Scanning: Barcode/QR Code API integration
